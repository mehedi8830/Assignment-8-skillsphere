## 📌 Project Purpose

SkillSphere is designed to help learners upgrade their skills through structured online courses. It provides a smooth user experience with authentication, protected routes, and interactive UI.

---

## ✨ Key Features

- 🔐 User Authentication (Email/Password + Google Login)
- 🏠 Responsive Home Page with Hero Section
- 📚 All Courses Page with Search Functionality
- 🔒 Protected Course Details Page
- 👤 My Profile Page (Update Name & Image)
- ⭐ Popular Courses Section (Top Rated)
- 📈 Trending / New Release Courses Section
- 👨‍🏫 Top Instructors Section
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 🔔 Toast Notifications
- ⚡ Loading Spinner on Data Fetch
- ❌ Custom Not Found Page
- 🔄 No reload crash (App Router handled)

---

## 🧱 Layout Structure

### 🔝 Navbar
- Logo (SkillSphere)

- Login / Register (if logged out)

### 🔻 Footer
- Contact Info
- Social Links
- Terms & Conditions
- Privacy Policy

---

## 📄 Pages & Routes

- `/` → Home
- `/courses` → All Courses
- `/courses/[id]` → Course Details (Protected)
- `/login` → Login
- `/register` → Register
- `/my-profile` → Profile (Protected)
- `/update-profile` → Update Info
